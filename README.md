# Group B_Recycling Website_Sustainability Society

This repository contains a web-based project for the Sustainability Society, including an interactive quiz and simple games to educate users about sustainability topics. The project is built with **HTML**, **CSS**, and **JavaScript**.

## Project Structure

- `index.html` – Home page with navigation to the quiz and game pages.
- `quiz1.html` – Interactive quiz page with sustainability questions.
- `game1.html` – Simple game page related to sustainability concepts.
- `style1.css` – CSS file that styles all the pages for layout, colors, fonts, and buttons.
- `script1.js` – JavaScript file for interactive elements such as quiz scoring and game logic.
- `Pictures/` – Folder containing all images used in the project (e.g., illustrations, icons, banners).

## Features

- **Home Page**: Serves as the main navigation hub for the quiz and game sections.
- **Quiz Page**: 
  - Multiple-choice questions on sustainability.
  - Tracks the user’s score.
  - Displays results at the end.
- **Game Page**:
  - Interactive mini-game for learning sustainability concepts.
  - User-friendly interface with visual feedback.
- **Responsive Design**: CSS ensures the pages display properly across different screen sizes.

## How to Use

1. Open `index.html` in a web browser.
2. Use the navigation links to go to:
   - The **Quiz Page** to test knowledge about sustainability.
   - The **Game Page** for interactive learning.
3. All images and styles are loaded from local files in the same folder or the `Pictures` directory.
